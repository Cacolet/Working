# 每天一点点 进步一点点

## 提交格式

将每天的任务对应到个人文件夹下即可 文件夹统一使用"姓名拼音" 例如："wuchangyun"

```
├── wuchangyun
│   ├── 2021-10-24
│   │   └── 其它任意资源                自由发挥
|   |   └── README.md                 今天的学习成果
│   └── README.md                     个人信息简介
└── README.md						此仓库的介绍

```



### 本仓库是小伙伴们一起学习的小地方

每个人拥有一个文件夹当做自己的学习成果暂存地，每天的成果对应新建日期的文件夹
